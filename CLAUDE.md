## Growth Pillars

### 1. Content Strategy
- Every video must target a searchable topic with clear audience demand
- Titles should be specific, curiosity-driven, and keyword-rich
- Each video should answer one focused question or demonstrate one concrete outcome
- Aim for 1-2 uploads per week minimum for consistent algorithmic visibility

### 2. Thumbnails
- High contrast, bold typography, one clear focal point
- A/B test thumbnail concepts before finalizing
- Faces or results perform better than abstract visuals — use real output screenshots where possible
- When building animated website demos, capture compelling visuals for thumbnails

### 3. SEO & Discovery
- Research keywords before scripting each video (YouTube Search, Google Trends, VidIQ, TubeBuddy)
- Include target keyword in: title, first 2 lines of description, and tags
- Use chapters/timestamps for every video over 5 minutes
- Pin a comment with key links and CTA within 30 minutes of publishing

### 4. Retention & Watch Time
- Hook in the first 15 seconds — show the end result immediately, then explain how to get there
- Cut all dead air, filler phrases, and re-introductions mid-video
- Use B-roll, code previews, and animated transitions to maintain visual engagement
- End with a strong CTA and tease the next video

### 5. Animated Website Projects as Content
- Every animated website or UI project built in this workspace is potential video content
- Document the build process for tutorials
- Ship demos as standalone web pages that can be linked in video descriptions (drives traffic + showcases skill)
- Embed channel links and CTAs in demo pages

### 6. Community & Engagement
- Reply to every comment in the first 24 hours after publishing
- Ask one specific question at the end of each video to prompt comments
- Use the Community tab for polls, WIPs, and engagement between uploads
- Collaborate with other creators in the web dev / creative coding niche

### 7. Analytics-Driven Iteration
- Review YouTube Studio analytics weekly: CTR, AVD (average view duration), traffic sources
- If CTR < 4%, test a new thumbnail or title
- If AVD drops below 40%, identify where viewers leave and cut or restructure that section
- Double down on video formats and topics that outperform
