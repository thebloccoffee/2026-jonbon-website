# Downloads Folder

Place your product files here. They are served securely through the `/download` endpoint
after payment verification — direct URL access returns 403 Forbidden.

## Expected files
- `lut_pack.zip`           — Cinematic LUT Pack
- `filmmaking_guide.pdf`   — Travel Filmmaking Guide
- `editing_workflow.zip`   — Creator Editing Workflow
- `creator_bundle.zip`     — Full Creator Bundle (all three above)

## Important
- Do NOT rename files without also updating the `PRODUCTS` map in `server.js`
- Keep this folder out of public git repositories if files are proprietary
